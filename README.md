### 🛠️ Quick Installation

#### **Termux (Android)**
If you're on the go, Termux is your best friend. Just run:
```bash
pkg update && pkg upgrade
pkg install python git ffmpeg -y
git clone https://github.com/itswill00/Ultroid
cd Ultroid
bash install-termux
```

#### **VPS / Local System**
For a professional deployment on your own server:
```bash
# Clone the repo
git clone https://github.com/itswill00/Ultroid.git && cd Ultroid

# Set up environment
python3 -m venv venv && source venv/bin/activate
pip install -U -r requirements.txt

# Start your journey
bash startup
```
